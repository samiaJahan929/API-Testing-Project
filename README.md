# DMoney API Project
# Introduction
DMoney is a transactional system that allows users to send money, make payments, and perform various other financial transactions.

# Technology Used
- Postman
- Newman
- NodeJs

# How to Run the Project
- Clone this project
- PUT .env file
- Give Following Commands :
  -  ``` npm i ```
  -  ``` npm test ```
